What Is MetaMask?

MetaMask is a self-custodial crypto wallet that runs as a browser extension and as a mobile app. It lets you:

  • Store and send cryptocurrencies such as ETH and many ERC-20 tokens.
  • Connect to decentralized apps (dApps) directly from your browser.
  • Manage NFTs and interact with Web3 games, DeFi platforms, and more.

Unlike centralized exchanges, MetaMask is non-custodial: you control your wallet through your Secret Recovery Phrase (seed phrase). The company behind MetaMask cannot access your funds or reset your wallet if you lose that phrase.

Before You Install: Requirements and Safety Checklist

Supported browsers and devices

As of today, MetaMask is available as:

  • Desktop browser extension: Google Chrome, Firefox, Brave, Edge, Opera.
  • Mobile app: iOS (iPhone/iPad) and Android.

Desktop Safari does not support a MetaMask extension. On Mac, you‚Äôll need a supported browser (for example, Chrome or Firefox) or you can use the MetaMask mobile app.

System and browser requirements

  • Use a relatively recent version of Windows, macOS, or Linux.
  • Update your browser (Chrome, Firefox, Brave, Edge, or Opera) to the latest version before installing the extension.
  • On mobile, keep iOS or Android up to date via system updates.

Security checklist before downloading MetaMask

  • Use only official download links. The safest option is to type https://metamask.io/download directly into your browser‚Äôs address bar instead of clicking random search results or ads.
  • Check the website URL: it must start with https:// and belong to metamask.io.
  • Verify the publisher in the extension store or app store: it should reference MetaMask / ConsenSys (names can vary slightly by store).
  • Avoid public Wi-Fi when first setting up your wallet. Use a trusted home or office connection.
  • Prepare secure storage (paper or metal) for your Secret Recovery Phrase. Never store it in screenshots, cloud notes, or email.

How to Download and Install MetaMask Extension on Desktop Browsers

In this section you‚Äôll learn how to install the MetaMask Chrome extension on desktop. The flow is almost identical for other supported browsers.

Step 1 ‚Äì Go to the official MetaMask download page

  1. Open your desktop browser (for example, Google Chrome).
  2. In the address bar, type https://metamask.io/download and press Enter.
  3. On the MetaMask download page, choose Browser Extension.

From this page, you‚Äôll see buttons or icons for each supported browser (Chrome, Firefox, Brave, Edge, Opera). Clicking your browser will redirect you to the official extension store listing.

Step 2 ‚Äì Install MetaMask extension on Google Chrome (Desktop)

Follow these steps specifically for the Chrome browser:

  1. On the MetaMask download page, click the button for Chrome. You‚Äôll be redirected to the official MetaMask listing on the Chrome Web Store.
  2. Double-check that:
    • The extension name is MetaMask.
    • The developer/publisher references MetaMask or ConsenSys.
    • The logo is the familiar orange fox.
    • There are many installations and reviews (fake clones usually have very low numbers).
  3. Click Add to Chrome.
  4. A pop-up will show the permissions MetaMask needs (for example, reading and modifying data on sites you visit). Review them, then click Add extension if you agree.
  5. Wait for Chrome to download and install the extension. You should see a confirmation message once MetaMask is added.
  6. Click the puzzle-piece icon in the Chrome toolbar (Extensions menu), find MetaMask, and click the pin icon to keep it visible next to your address bar.

Now the MetaMask fox icon should appear in your Chrome toolbar. Click it any time to open your wallet once setup is complete.

Step 3 ‚Äì Install MetaMask in other desktop browsers

The steps are very similar in other supported browsers:

Brave

  1. Open Brave and go to https://metamask.io/download.
  2. Select Browser Extension > Brave. Brave uses the Chrome Web Store, so you‚Äôll see a similar listing.
  3. Click Add to Brave (or equivalent) and confirm installation.
  4. Pin the MetaMask icon from Brave‚Äôs extension menu.

Microsoft Edge

  1. Open Edge and go to https://metamask.io/download.
  2. Select Browser Extension > Edge. You‚Äôll be redirected to the Edge Add-ons store.
  3. Click Get, then confirm by clicking Add extension.
  4. Pin MetaMask from the extensions menu for quick access.

Firefox

  1. Open Firefox and go to https://metamask.io/download.
  2. Click the Firefox option to open the Mozilla Add-ons page.
  3. Click Add to Firefox, then confirm in the permissions dialog.
  4. Restart Firefox if prompted and ensure the MetaMask icon is available in your toolbar.

Opera

  1. Open Opera and go to https://metamask.io/download.
  2. Select Opera from the extension options.
  3. Follow the store prompts to install and enable the MetaMask extension.

Step 4 (Advanced) ‚Äì Manual installation of MetaMask extension

This is mainly for advanced users and developers. MetaMask can also be installed manually as an ‚Äúunpacked‚Äù extension:

  • Download a release package from official MetaMask repositories.
  • Unzip the file to a folder on your computer.
  • In Chrome, open chrome://extensions, enable Developer mode, and click Load unpacked.
  • Select the unzipped MetaMask folder to load it as a local extension.

If you‚Äôre not sure you need this, stick to the normal installation from the official extension store‚Äîit's safer and easier.

How to Download and Install MetaMask on Mobile (iOS & Android)

MetaMask Mobile is a full wallet with a built-in browser that lets you connect to dApps directly from your phone. The safest way to get it is still via the official website or the official app stores.

Step 1 ‚Äì Open the official download page or app store

  • On your phone, open a browser and go to https://metamask.io/download.
  • Tap Mobile App and then choose iOS or Android. This will redirect you to the correct listing in the App Store or Google Play.

You can also search manually in the App Store or Google Play for ‚ÄúMetaMask‚Äù, but make sure you choose the official app with the fox logo and a large number of installs and reviews.

Step 2 ‚Äì Install MetaMask on iOS (iPhone & iPad)

  1. Open the App Store on your iPhone or iPad.
  2. Search for MetaMask (or use the link from the official website).
  3. Confirm that:
    • The app name includes ‚ÄúMetaMask‚Äù.
    • The publisher is legitimate (MetaMask / ConsenSys).
    • The icon is the orange fox head.
  4. Tap Get (or the download cloud icon), then authenticate with Face ID, Touch ID, or your Apple ID password.
  5. Wait for the app to download and install, then tap Open.

Step 3 ‚Äì Install MetaMask on Android

  1. Open the Google Play Store on your Android device.
  2. Search for MetaMask, or use the direct link from the official MetaMask download page.
  3. Verify the following:
    • The app name is MetaMask ‚Äì Crypto Wallet or similar.
    • The developer is MetaMask or ConsenSys.
    • The logo is the orange fox.
    • The download count is very high (millions of installs) and there are many reviews‚Äîclones usually don‚Äôt have this.
  4. Tap Install and wait for the installation to complete.
  5. Tap Open to launch MetaMask.

How to Set Up Your MetaMask Wallet After Installation

Once the MetaMask extension (desktop) or app (mobile) is installed, you‚Äôll see an onboarding flow. The screens on desktop and mobile are very similar.

Option 1 ‚Äì Create a new MetaMask wallet

  1. Click or tap Get Started.
  2. Select Create a new wallet.
  3. Read and accept the Terms of Use and any privacy prompts.
  4. Create a strong password for this device:
    • Use at least 12 characters.
    • Include a mix of upper/lowercase letters, numbers, and symbols.
    • Do not reuse passwords from other websites.
  5. MetaMask will show you your Secret Recovery Phrase (usually 12 words).
    • Write these words down on paper (or engrave/print them offline).
    • Keep them in a place only you can access.
    • Never share the phrase with anyone, not even ‚Äúsupport agents‚Äù or friends.
    • Never store it in email, cloud drives, screenshots, or messaging apps.
  6. Confirm your phrase by selecting or typing the words in the correct order.
  7. After confirmation, your wallet will be created and you‚Äôll see your first account with a public address.

Option 2 ‚Äì Import an existing wallet

If you already have a MetaMask wallet or another compatible wallet, you can import it using the same Secret Recovery Phrase:

  1. On the welcome screen, click or tap Import wallet.
  2. Enter your 12-word Secret Recovery Phrase carefully, in the correct order.
  3. Create a strong local password for this device.
  4. Finish the setup. Your existing accounts and balances will appear. In some cases, you may need to re-add custom networks or tokens.

Important: Anyone who gets your Secret Recovery Phrase can fully control your wallet from any device. Treat it like the master key to all your funds.

How to Use the Same MetaMask Wallet on Desktop and Mobile

You can use one MetaMask wallet across multiple devices: for example, Chrome extension on your laptop and the MetaMask app on your phone. This works because your wallet is tied to your Secret Recovery Phrase, not to a single device.

Method 1 ‚Äì Import the same Secret Recovery Phrase

  1. On the device where MetaMask is already set up, make sure you have your Secret Recovery Phrase written down safely.
  2. On the new device (desktop or mobile), install MetaMask as described above.
  3. Choose Import wallet on the welcome screen.
  4. Enter the same Secret Recovery Phrase and create a local password.
  5. Your accounts and balances will appear on the new device after synchronization with the blockchain.

Changes like sending funds or interacting with dApps are reflected on all devices because they‚Äôre on the blockchain, not stored in the app itself.

Method 2 ‚Äì Backup & Sync features

MetaMask also offers backup and sync features (such as cloud-backed sync or account/contact sync) that can keep things like account names and contacts aligned across devices. You can manage these options in Settings > Backup and sync on supported versions of the extension and mobile app.

If you‚Äôre not comfortable with cloud-based options, you can rely solely on the Secret Recovery Phrase method.

Common Installation Problems and How to Fix Them

1. I can‚Äôt find the MetaMask icon after installing

  • In Chrome and other Chromium browsers, click the puzzle icon (Extensions) and pin MetaMask.
  • In Firefox, open the Add-ons menu, confirm MetaMask is enabled, and drag it into the toolbar if necessary.

2. The store says MetaMask is ‚Äúnot compatible‚Äù with my browser

  • Update your browser to the latest version.
  • Switch to a fully supported browser: Chrome, Firefox, Brave, Edge, or Opera.
  • If you‚Äôre using Safari on desktop, install one of the supported browsers or use MetaMask Mobile instead.

3. I installed MetaMask but nothing happens when I click it

  • Disable and re-enable the extension in your browser‚Äôs extension manager.
  • Close and reopen your browser.
  • Try temporarily disabling other extensions that might conflict with MetaMask (especially privacy or script-blocking tools).
  • If problems persist, consider removing MetaMask and reinstalling it from the official listing.

4. High CPU, RAM, or disk usage

Crypto wallet extensions can occasionally cause performance issues, especially if you use many dApps or hold a complex on-chain history.

  • Make sure MetaMask is updated to the latest version.
  • If you notice unusual disk activity or resource use, you can temporarily disable or remove the extension and use the mobile app instead while you investigate.
  • Regularly review official MetaMask announcements regarding performance or known bugs.

Security Best Practices When Using MetaMask

Installing MetaMask correctly is only the first step. Keeping your funds safe requires ongoing good habits.

  • Never share your Secret Recovery Phrase. No legitimate support agent, website, or dApp will ever need it.
  • Beware of fake websites. Always double-check URLs before connecting your wallet or entering sensitive information.
  • Use a hardware wallet (Ledger, Trezor, etc.) together with MetaMask for larger balances.
  • Lock MetaMask when you‚Äôre not using it, especially on shared computers. Use the lock option in the account menu.
  • Keep backups of your Secret Recovery Phrase in at least two secure physical locations.
  • Update regularly. Keep your browser, operating system, and MetaMask up to date to benefit from the latest security fixes.
  • Be cautious with approvals. When a dApp asks to spend your tokens, confirm that you trust the site and understand what you‚Äôre approving.

Remember: if someone gets your Secret Recovery Phrase, they can import your wallet into their own MetaMask extension or mobile app and take full control over your assets.

FAQ: Downloading and Installing MetaMask Extension

Is MetaMask free to download and install?

Yes. The MetaMask extension for Chrome and other browsers, as well as the mobile apps for iOS and Android, are free to download and install. You only pay network fees (like gas on Ethereum) when sending transactions.

Can I install MetaMask on Safari?

There is no official MetaMask extension for desktop Safari. On Mac, use a supported browser such as Chrome, Firefox, Brave, Edge, or Opera, or install the MetaMask mobile app on your iPhone or iPad.

Is it safe to use MetaMask on more than one device?

Yes, you can use the same MetaMask wallet on multiple devices by importing it with the same Secret Recovery Phrase. Make sure each device is secure (strong passwords, no malware, updated software). If any device is compromised, consider that wallet compromised as well.

How do I update the MetaMask Chrome extension?

Most browsers update extensions automatically. To force an update in Chromium-based browsers, go to the extensions page (for example, chrome://extensions), enable Developer mode, and click Update. On Firefox, you can check for updates in the Add-ons manager.

What happens if I lose my phone or my computer?

If you still have your Secret Recovery Phrase, you can simply install MetaMask on a new device and use Import wallet to restore access. If you lose both your device and your Secret Recovery Phrase, your funds cannot be recovered.

How do I know if an extension or app is the real MetaMask?

Always start from https://metamask.io/download, check that the publisher and logo look correct, and verify that the extension or app has many downloads and reviews. If anything looks suspicious, do not install it.

Summary: To safely install the MetaMask extension on desktop (especially on Chrome) and the mobile apps on iOS/Android, always use the official MetaMask download page, verify the publisher in the store, carefully store your Secret Recovery Phrase offline, and keep your software updated. Done properly, your MetaMask setup will be ready for Web3 in just a few minutes.
